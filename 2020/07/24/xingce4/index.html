行测笔记：判断推理（四）逻辑判断

发表于 2020-07-24 更新于 2020-07-25 分类于 行测笔试 阅读次数： Valine： 本文字数： 1.1k 阅读时长 ≈ 1 分钟

一、翻译推理

题目特征

  1. 题干和选项中存在明显的逻辑关联词
  2. 提问方式为：可以推出/不能推出

解题思维

  1. 先翻译
  2. 再推理

（1）翻译规则之“前$\rightarrow$后”

  1. 若…则
  2. 只要..就
  3. 所有…都
  4. 为了…一定
  5. …是…的充分条件

（出现如果/就/都/一定）

（2）推理规则之“逆否等价”

符号表示：$A\rightarrow B = -B\rightarrow -A$

文字表示:肯前比肯后、否后必否前

否前、肯后无必然结论

（3）翻译规则之“后推前”

  1. 只有…才…
  2. 不…不…
  3. 除非..否则不…
  4. …是…的基础/假设/前提/关键
  5. …是…的必要/必不可少条件

“否则不”是一个整体
6. 基础/前提/关键;不可缺少/必不可少/必要条件

谁必不可少，谁在箭头后

都遵循递推原则：$A\rightarrow B，B\rightarrow C $可得$A\rightarrow B\rightarrow C$

（4）翻译规则之“且、或”

A且B：二者同时成立

  1. 和
  2. 并且
  3. 即…又…
  4. 不仅…而且..
  5. …但是…

A或B：二者至少一个成立

  1. 或者
  2. 或者…或者…
  3. 至少有一个

（5）推理规则之“否一推一”

“或”的翻译：否1$\rightarrow$1

“或”关系为真，否定一项可以得到另一项

题干中有确定信息，可由确定信息出发，排除法做题

（6）推理规则之“德*摩根定律”

  1. -(A且B) = -A或-B
  2. -(A或B) = -A且-B

“且”关系常考考点：缺一不可

A$\rightarrow$B且C，可以得到：-B$\rightarrow$-A/-C$\rightarrow$-A

（7）推理方式

提问方式：

一下哪项中的推理方式/结构与题干中的推理方式/结构相同

重结构、轻内容——字母代入做题

不考虑推理是否正确，只考虑与题干是否一致

二、组合排列

  1. 两组及以上对象
  2. 对象之间的关系

（1）排除法、代入法

  1. 排除法：读一句，排一句
    • 快速找到“谁”是“谁”
    • 快速确定“谁”不是“谁”

组合排列常用思维——谁一定不是谁

  1. 代入法：假设选项正确，代入题干验证是否符合题意
    • 题干条件确定优先排除；题干条件不确定代入
    • 设问中有“可能”、“不可能”考虑代入
      只猜中一半（一对一错）——混搭来破题

（2）辅助技巧

  1. 最大信息（题干条件中出现次数最多的信息）
    以此作为推理起点
  2. 符号：”>”、“<”、“=”
    往往涉及年龄、成绩、收入、身高等大小比较

比较大小技巧——极值做排除

  1. 画表格（4个及以上信息）
    先填入确定信息

三、特殊题型——材料题

  1. 与非材料题解题方式一致——一则材料、多种技巧
  2. 通过材料直接推出来的结论可以直接应用于所有题目

当且仅当表示充要条件，即A$\rightarrow$B，且B$\rightarrow$A
