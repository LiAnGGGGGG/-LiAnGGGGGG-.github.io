行测笔记：判断推理（三）定义判断

发表于 2020-07-23 分类于 行测笔试 阅读次数： Valine： 本文字数： 467 阅读时长 ≈ 1 分钟

解题思维：

  1. 看提问——看清“属于/符合”,“不属于/不符合”定义等
  2. 看题干——识别有效信息：找准关键词、关键句
  3. 看选项——纠结时选项对比择优

一、快速识别有效信息

（1）关键词——主体、客体

  1. 主体——行为、活动的出发者
  2. 客体——行为、活动作用的对象
  • 不是每题一定都有主题和客体，但是出现了要重点看
  • 主客体不符合的选项一定不符合定义
  • 范围越小、越明确的主/客体，越容易成为考点

常考主客体：

  1. 行政机关
  2. 全力机关
  3. 司法机关
  4. 事业单位
  5. 社会组织

（2）句式引导词——引导的内容重点看

  1. 方式：通过/利用…
  2. 目的：以/以达到…
  3. 原因：因为/由于…
  4. 结果：导致/从而/使得…
  5. 时间：当…时/在…情况下

（3）优先看到定义所在的那句话的句号和补充说明

（4）多定义：做题方法与单定义题目一致

  1. 问啥看啥：看清题干问的是哪个定义，重点看目标定义
  2. 比较定义：纠结两个选项时，再与其他定义比较

二、小技巧

（1）拆词

从定义本身找信息，如果定义名称有明确的限定语并简单易懂，可利用定义词快速破题

（2）同构选项排除法

意思相似、结构相似的选项排除

  1. 题干或下选项能够读不懂
  2. 同构选项出现，可以一起排除
  3. 同构有风险，使用需谨慎…
